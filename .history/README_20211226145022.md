# infosearch

## Project setup
```
npm install
```
## download dependency library
```
npm install axios
npm install element-ui
npm install router
```
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
### note

